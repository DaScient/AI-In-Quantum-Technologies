# Interactive Learning Paths

These thematic tutorials cut across multiple chapters of *AI in Quantum
Technologies*. Each is a self-contained study combining prose (`README.md`) with a
runnable script (`walkthrough.py`) that uses only the dependency-light core
library, so every path executes offline:

```bash
python tutorials/03_qec_decoder_challenge/walkthrough.py
```
