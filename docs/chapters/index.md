# The Textbook

*AI in Quantum Technologies: Theory, Applications, Practice, and Society* is
organised into eight parts that build, step by step, the conceptual and
practical machinery required to apply artificial intelligence across the quantum
technology landscape. Each chapter pairs rigorous exposition with executable
code in the [`aiquantum`](../api.md) library, so that every idea can be run,
inspected, and extended.

## Structure

| Part | Theme | Chapters |

| **IV** | AI for Quantum Matter | 10–12 |

| **VI** | Quantum Machine Learning & Hybrid Systems | 16–18 |
| **VII** | Ethics, Policy, and Societal Impact | 19–21 |

## How each chapter is built

Every chapter follows a consistent pedagogical arc:

1. **Learning objectives** — what the reader will be able to do.
2. **Motivation** — why the problem matters scientifically and societally.
3. **Theory** — the core concepts, developed in full prose with key equations.
4. **From theory to code** — the corresponding `aiquantum` functions and API
   endpoints, with runnable examples.
5. **Summary and exercises** — consolidation and extension.
6. **Further reading** — pointers to the primary literature.
