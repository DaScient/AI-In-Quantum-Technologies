# Contributing

We welcome contributions from the global quantum and AI communities. The
authoritative guidelines live in
[`CONTRIBUTING.md`](https://github.com/DaScient/AI-In-Quantum-Technologies/blob/main/CONTRIBUTING.md)
at the repository root. In brief:

- **Report bugs** with a minimal reproducible example.
- **Submit code** via fork → branch → pull request; all code must include tests
  and pass CI (`ruff check` and `pytest`).
- **Improve documentation** by editing files under `docs/`.
- Keep the core library dependency-light; heavier back-ends belong in optional
  extras, notebooks, and tutorials.

We are especially keen on contributions that add new quantum backends, provide
multi-lingual notebook translations, or expand the ethical auditing framework.
